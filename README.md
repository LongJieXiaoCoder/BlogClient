# BlogClient
博客园iOS App

## Api 文档
- https://api.cnblogs.com/help

## 工程结构
### 首页
    - 博文列表
    - 最新博文
    - 博文详情
    - 评论
### 新闻
    - 推荐新闻
    - 新闻列表
    - 新闻内容
    - 热门新闻
    - 本周热门
    - 新闻评论
### 闪存（类似沸点）
    - 闪存列表 
    - 闪存内容
    - 发布闪存
    - 删除闪存
    - 闪存评论
    - 删除删除评论
    
### 知识库
    - 知识库列表
    - 文章内容
### 我的
    - 个人博客信息
    - 个人博客列表
    - 我的收藏(支持删除)

### 其他功能
    - 分享
    - 登录
    - 其他第三方内容
